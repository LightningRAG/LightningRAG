# 贡献者公约行为准则

**English:** [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md)

## 我们的承诺

为促进开放、友好的环境，我们作为贡献者与维护者承诺：无论年龄、体型、残障、族裔、性征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教信仰或性取向如何，参与本项目和社区的经历都不应受到骚扰。

## 我们的标准

有助于营造积极环境的行为包括：

* 使用友好、包容的语言  
* 尊重不同的观点与经历  
* 虚心接受建设性批评  
* 关注对社区最有利的事情  
* 对其他社区成员保持同理心  

参与者不应出现的行为包括：

* 使用带有性意味的语言或图像，以及不受欢迎的性关注或性示好  
* 挑衅、侮辱/贬损性评论，以及人身或政治攻击  
* 公开或私下骚扰  
* 未经明确许可公布他人的私人信息（如实体或电子地址）  
* 在专业场合中可能被合理认为不适当的其他行为  

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并对任何不当行为采取恰当、公平的纠正措施。

维护者有权删除、编辑或拒绝与本行为准则不符的评论、提交、代码、Wiki 编辑、Issue 等贡献，或因其他不当、威胁、冒犯或有害行为暂时或永久禁止任何贡献者。

## 适用范围

本准则适用于项目空间内，以及个人在公开场合代表本项目或其社区时。代表项目或社区的情形包括：使用官方项目邮箱、通过官方社交媒体发布内容，或作为指定代表参加线上或线下活动。维护者可进一步定义或说明「代表」的含义。

## 执行

如遭遇辱骂、骚扰或其他不可接受行为，可向项目团队发邮件至 **newfarry@126.com** 报告。所有投诉将被审查与调查，并给出被认为必要且适当的回应。项目团队有义务对报告人保密。具体执行政策可另行发布。

未善意遵守或执行本行为准则的维护者，可能由项目领导层决定面临暂时或永久后果。
